LG.com HS Dashboard package

Updated in v17:
- lg-com-status-dashboard now automatically searches for a local Excel file named 신호등data-MMDD.xlsx in the same folder.
- Example: lg-com-status-dashboard/신호등data-0701.xlsx -> week-tab/data-week displays Jul 01.
- Future dates work the same way: 신호등data-1205.xlsx -> Dec 05.
- The date in the filename is used for the .week-tab text and data-week value.
